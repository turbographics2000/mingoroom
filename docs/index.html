「みんなでゴルフ待合所 (仮題)」について

「みんなでゴルフ待合所 (仮題)」は拡張機能で動作するシステムとなっています。ですので、拡張機能をインストールしなければなりません。
拡張機能はChromeの拡張機能となっていますので、対応ブラウザーはChromeのみ、また、対応デバイスはPCのみとさせていただきます。
初めに
「みんなでゴルフ待合所 (仮題)」は経費0にするため、サーバーのないシステムとなっています。
そのため、普通のシステムとは違いちょっと面倒くさいシステムとなっています。
次のステップから面倒くさいところの説明をしますので、お読みになりご理解のほどよろしくお願いいたします。
面倒くさいところ その1/2

アカウント登録

アカウント登録は必須とさせていただきます。
少し手間がかかるTwitterアカウントを利用したなんちゃって認証による登録となっているため、Twitterアカウントも必須となります。
サーバーがないため登録情報は、ローカルに保存されます。
面倒くさいところ その2/2

待合室作成

待合室を作成すると、アカウント情報と同様にローカルに保存されます。
他のユーザーの画面に自分が作成した待合室を表示させるには、「みんなでゴルフ待合所 (仮題)」を開いておかなければなりません。
アカウント登録
１．まず、Twitterのアカウントのユーザー名を以下の認証キーに変更します。
２．みんゴルの名前とTwitterのIDを入力し、登録ボタンをクリックしてください。
みんゴル名
Twitter ID
ユーザー登録行いました。Twitterのユーザー名は元に戻して構いません。
